꺽새 link rel="stylesheet" href="경로" 꺽새 를 써줘야한다
  1. 폰트의 색을 빨간색으로 하고 싶을때
    이 리스트 태그에 내가 임의로 red라는 클래스를 꺽새 li class="red" 꺽새 이렇게 넣어주고
    css파일에서 .red { color: #원하는색; } 이렇게 넣어준다
    저 점표시가 클래스라는뜻임 li에 id로 넣어준 경우 (꺽새li id="red"꺽새) #red {color: #색코드 ;} 이렇게씀

  2. 배경의 색을 초록색으로 바꾸고 싶을 때 머시기 { background-color: #색코드 ;}>

  3. 폰트의 크기를 바꾸고 싶을 때
    css파일에 내가 폰트를 바꾸고 싶은 부분을 정의하거나 (클래스나 아이디나.. 클래스의 경우 꺽새안의 html태그 뒤에 class="머시기" 이렇게 써줘야함)
    태그명을써주고 {속성:속성값;}으로입력
    ex)
    • h2 { color: #ff0000;}
    • .red { font-size: 25px;}


  4. 그룹선택자--h1이면서 클래스red인애 --> h1.red { } //////// .headline.selected.red { } (저런클래스다가진걔한테 어떤걸하겠다)
    ul>li, thead>tr>th, h3.red {color: red; } --> 저 세개가 색이 똑같다면 이런식으로도 쓸수있다

    ex) p.red { background-color: #색상코드 ;} --> p태그이면서 클래스가 red인 녀석에게 배경색을주겠다
    li.red { font-size: ?? px ;} --> li 태그이면서 클래스가 red인 녀석에게 폰트사이즈를 바꾸겠다

  5. 속성 선택자 input[type="text"] { }
    a[target] {background-color: #ff0000;}
    a태그에 타겟을 속성으로가진애들 배경색 빨강으로바꿔라~~

    a[target="_blank"] {background-color:#색코드 ;}
    --> a태그이면서 타겟=_blank 을 속성으로 가진애들 배경색 바꿀거다

    tbody>tr>td a[href] {background-color: #ff0000;}
    바로밑에 이거 바로밑에 이거에 href갖고있는애들은저걸로바꿔~~
